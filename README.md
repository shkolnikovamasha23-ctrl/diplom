# EduSim

EduSim — учебный веб-симулятор на Flask. Проект моделирует прохождение образовательных сценариев, где студент принимает решения, видит последствия и получает итоговый отчет.

## Что изменено

Проект переработан так, чтобы отличаться от обычного теста:

- добавлены этапы сценария;
- добавлены ветвления и переходы;
- добавлено состояние симуляции: время, бюджет, риск, качество, регламент;
- файлы используются как артефакты этапов;
- итог зависит от всей истории действий;
- добавлена история решений и итоговое заключение.

## Демо-доступ

- admin / admin123
- teacher / teacher123
- student / student123

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

Или через Docker Compose:

```bash
docker compose up --build
```

## Структура

- `edusim/database.py` — схема БД, миграции, демонстрационные данные.
- `edusim/routes_cabinet.py` — личный кабинет и логика симулятора.
- `edusim/templates/cabinet/run_scenario.html` — прохождение сценария.
- `docs/requirements_mapping.md` — соответствие требованиям.
- `docs/test_plan.md` — функциональное и пользовательское тестирование.
- `docs/business_processes.md` — описание процессов и экономики.
