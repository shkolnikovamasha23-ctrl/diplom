# EduSim — образовательный симулятор

Веб-сайт разработан для темы ВКР «Разработка образовательного симулятора».

## Запуск без Docker

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Запуск через Docker

```bash
docker compose up --build
```

Сайт будет доступен по адресу `http://localhost:5000`.

## Что реализовано

- Flask-приложение на Python с корневым `main.py`.
- Современный адаптивный дизайн.
- Не менее 10 публичных страниц, верхнее и боковое меню, хлебные крошки.
- Форма обратной связи.
- SQL-база SQLite с более чем 10 таблицами.
- 3 роли пользователей.
- Панель администратора с 5+ страницами.
- Личный кабинет пользователя с 5+ страницами.
- Загрузка файлов в файловую систему.
- Формирование DOCX и XLSX.
- Справка с ФИО автора.

## Структура

- `edusim/routes_public.py` — публичные страницы.
- `edusim/routes_auth.py` — регистрация и вход.
- `edusim/routes_cabinet.py` — личный кабинет и симулятор.
- `edusim/routes_admin.py` — административная панель.
- `edusim/database.py` — схема БД и демонстрационные данные.
- `edusim/documents.py` — генерация DOCX/XLSX.
